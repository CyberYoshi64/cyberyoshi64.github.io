Custom Firmware

This application pack is designed to work using custom firmware (“CFW”).
CFW allows programs (such as “homebrew” and personal copies of applications) to run on the Nintendo 3DS system without approval by console manufacturers.

For more information, please refer to the following website:
https://3ds.hacks.guide/

Do you have issues with custom firmware on your system? Ask for help in the Nintendo Homebrew Discord server:
https://discord.gg/MWxPgEp

Luma 3DS CFW

This application pack makes use of extra features provided by Luma 3DS, a custom firmware for the family of Nintendo 3DS systems.
