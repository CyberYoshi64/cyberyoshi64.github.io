SmileBASIC-CYX is a modpack designed to enhance the SmileBASIC environment.

From allowing you to easily specify a custom server to moving your projects to the SD Card for easy importing/exporting, I hope CYX might be useful to you.

Launcher & Updater

This application is used to launch and update the modpack. When the console is connected to the internet, it checks for updates automatically and informs you when you're ready to download the latest updates.

Don't have CYX installed yet? Fear not! When CYX is not detected, you can install the necessary components and get your CYX instance up and running.

About Save Data

SmileBASIC-CYX seperates its own save data to ensure the extra settings and workspace does not collide with the vanilla SmileBASIC instance.

When CYX's save data doesn't exist, you can copy the entire save data of a SmileBASIC instance to CYX.

Copying the save data + entirety of extra data can take a long time to complete. Interrupting the copy process can result in broken BASIC projects whereas files will be missing or corrupted. Be careful!
