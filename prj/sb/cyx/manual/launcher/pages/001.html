Thank you for your interest in SmileBASIC-CYX, a game plugin for the Nintendo 3DS application “SmileBASIC”.

Custom Firmware

This application is designed to work using custom firmware (“CFW”).
CFW allows programs (such as “homebrew”) to run on the Nintendo 3DS system without authori­zation by console manu­facturers.

For more information, please refer to the following website:
https://3ds.hacks.guide/

CTRPluginFramework

This plugin is made with CTRPluginFramework, a C++ framework for 3GX plugins.
