SmileBASIC-CYX

SmileBASIC eXtensions is a plugin for SmileBASIC 3.

Powered by CTRPluginFramework, it makes using SmileBASIC 3 easier to use with external tools as well as adding some features to the BASIC environment.

Current features

  • Moving save data to the main SD Card (sdmc:/PTC3PLG)
  • Dark theme (W.I.P.)
  • Support for SBServer
  • Main SD Card access in BASIC programs (W.I.P.)
  • Miscellaneous 3DS information access (W.I.P.)
  • Data rescue on application crash
